# RubberDoc for Android

RubberDoc은 러버독이라 발음하며 **[꽥꽥 디자인 시스템](https://quackquack.duckie.team/android/)의 전문화된 문서를 제공**합니다.

꽥꽥은 Modifier 방식의 디자인 시스템이며, UI 플러그인 또한 제공합니다. 따라서, 기존 꽥꽥의 문서화 방식이었던 [Dokka](https://kotlinlang.org/docs/dokka-introduction.html)를 이용하기엔
`Component + Modifier + Plugin + Usage` 이 모든 니즈를 한눈에 파악하기 어렵다고 판단하여 꽥꽥에 전문화된 문서화 엔진을 RubberDoc이라는
이름으로 새롭게 개발하기로 결정하였습니다.

러버독은 꽥꽥 문서화를 전문적으로 하도록 만들어졌기에 외부 프로젝트엔 적합하지 않을 수 있습니다.
다만, 문서화 엔진 개발을 처음 시도하거나 독자적인 문서화 엔진 도입을 고민하시는 분께 도움이 됐으면 해서 public으로 진행합니다.

## Under the Hood

러버독은 [MDX](https://mdxjs.com/) 문서를 생성하며 [Docusaurus](https://docusaurus.io/)에 호스팅된다는 전제로 개발됩니다.
러버독의 작동 과정은 [architecture](architecture.mdx) 문서를 참고하세요.
